# 📑 Driver & Vehicle Licensing Department (DVLD) System

### 🚀 Overview
The **DVLD System** is a comprehensive desktop application designed to manage the processes of driver and vehicle licensing. This project demonstrates the implementation of professional software design patterns and a clean architectural approach.

### 🏗️ Software Architecture
The system is built using the **3-Tier Architecture** to ensure separation of concerns:
1.  **Presentation Layer (UI):** Built using **C# Windows Forms**.
2.  **Business Logic Layer (BLL):** Handles validation and business rules.
3.  **Data Access Layer (DAL):** Responsible for SQL Server database interaction using ADO.NET.

### 🛠️ Tech Stack
* **Language:** C#
* **Framework:** .NET Framework / WinForms
* **Database:** MS SQL Server
* **Design Patterns:** 3-Tier Architecture, Singleton

### 🌟 Key Features
* **User Management:** Secure login and role-based access control.
* **Applications:** Handling license renewals and replacements.
* **Tests Management:** Scheduling Vision, Written, and Street tests.
* **International Licenses:** Issuing international permits.
* **Detained Licenses:** Managing fines and releases.
